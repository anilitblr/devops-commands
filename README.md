# DevOps Command Reference

A quick reference guide for common DevOps tools and commands.

## Available Documentation

| Tool | Description | Commands |
|------|-------------|----------|
| [Git](docs/git.md) | Distributed version control system | 22 commands |
| [Docker](docs/docker.md) | Container platform | 60+ commands |

## Git

Version control commands organized by workflow:

- **Start a Working Area** - clone, init
- **Work on Current Change** - add, mv, restore, rm
- **Examine History and State** - bisect, diff, grep, log, show, status
- **Branch and History** - branch, commit, merge, rebase, reset, switch, tag
- **Collaborate** - fetch, pull, push

[View Git Documentation →](docs/git.md)

## Docker

Container management commands organized by category:

- **Common** - run, exec, ps, build, pull, push, images, login, logout, search
- **Container** - attach, commit, cp, create, logs, start, stop, rm, restart
- **Image** - history, import, inspect, load, rmi, save, tag
- **Network** - network create/ls/rm/connect/disconnect/inspect/prune
- **Volume** - volume create/ls/rm/inspect/prune
- **Compose** - up, down, ps, logs, build, exec, restart, pull
- **System** - system df/prune/info, events
- **Swarm** - init, join, leave, update

[View Docker Documentation →](docs/docker.md)
